PROJECT DESCRIPTION

What are we creating?

We are creating a database management system (DBMS) for the Blue Garden Botanical Library. The Library has over 800,000 works and manuscripts, and provides access to several thousand electronic titles including journals, ebooks, and online resources.

Why?

The Blue Garden Botanical Library database management system will drastically improve and facilitate the catalogue search and overall library experience for both users and employees alike. As the Library has yet to implement a database management system, the introduction of this system will be making great changes to the way the library functions. The DBMS will allow library employees to enter new resources, make changes to existing resources, and keep track of resources being borrowed without the added burden that a manual system causes.
